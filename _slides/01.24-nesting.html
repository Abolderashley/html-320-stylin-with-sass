nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;
  }

  li { display: inline-block; }

  a {
    display: block;
    padding: 6px 12px;
    text-decoration: none;
  }
}

Example: http://codepen.io/tamouse/pen/yJBVdZ

Sass allows the nesting of CSS so you can clearly tell which rules apply to which parent, without endless repetition.

Sass version:

    
nav
  ul
    margin: 0
    padding: 0
    list-style: none

  li
    display: inline-block

  a
    display: block
    padding: 6px 12px
    text-decoration: none
