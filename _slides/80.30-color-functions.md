### {{page.title}} Example

Start with SCSS:

{% highlight scss %}
$background: #f3f3f3;
$text_color: darken($background, 30%);
$accent_color: lighten($text_color, 10);
body {
  color: $text_color;
  background: $background;
  a {
    color: $accent_color;
  }
}
{% endhighlight %}
