## {{page.title}}

Along with looping, Sass provides a way to branch based on the
evaluation of an expression.

* `@if`
* `@else`
* `@else if`


</section>

<section>

#### {{page.title}} - SCSS

{% highlight scss %}
$type: monster;
p {
  @if $type == ocean {
    color: blue;
  } @else if $type == matador {
    color: red;
  } @else if $type == monster {
    color: green;
  } @else {
    color: black;
  }
}
{% endhighlight %}

</section>

<section>

#### {{page.title}} - CSS

Yields:

{% highlight css %}
p {
  color: green; }
{% endhighlight %}
