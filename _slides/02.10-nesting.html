--- layout: slide title: "Nesting" ---

Sass input:

header {
    color: black;
    nav {
        background: red;
        a { color: white; }
    }
}

CSS output:

header { color: black; }
header nav { background: red; }
header nav a { color: white; }

Sass dialect:

header
  color: black
  nav
    background: red
    a
      color: white
