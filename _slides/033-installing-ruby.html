--- layout: slide title: "Installing Ruby" ---

  • Windows users: go to RubyInstaller.org and follow the instructions to install both Ruby and the Ruby DevKit (under Add-Ons).

  • Mac users: You may have it a little easier, as Ruby may already be installed. Try this: 

    	  
ruby --version
    If the value returned is less than 2.0.0, you will need to install a newer version. Ask a TA for assistance.
